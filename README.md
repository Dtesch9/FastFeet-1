## BACKEND


##### Instalando as dependências

```sh
yarn install
```

##### Rodando as migrations do banco de dados

```sh
yarn sequelize db:migrate
yarn sequelize db:seed:all
```

##### Rodando a aplicação local (dev)

```sh
yarn dev
```
